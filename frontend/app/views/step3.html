Ihr persönlicher Code

Bitte erstellen Sie Ihren persönlichen Code. Dieser Code besteht aus einer Kombination von Buchstaben und Zahlen, die außer Ihnen niemandem bekannt ist, für Sie persönlich jedoch leicht wieder herzuleiten ist.

Der persönliche Code setzt sich wie folgt zusammen:

  • Die ersten beiden Buchstaben des Vornamens Ihrer Mutter.
  • Die ersten beiden Buchstaben Ihres Geburtsortes.
  • Die ersten beiden Zahlen des Geburtstages Ihres Vaters.
  • Die ersten beiden Zahlen Ihres Geburtstages.

Beispiel:
Ihre Mutter heißt Helga, Sie sind in Frankfurt geboren, Ihr Vater ist am 16.8.1950 geboren, Sie sind am am 02.03.1980 geboren.

Ihr persönlicher Code lautet somit HEFR1602.
