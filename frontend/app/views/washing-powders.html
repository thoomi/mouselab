Wie sehen die Waschmittel aus?

Im Verlauf der Studie werden Sie vor verschiedene Entscheidungsaufgaben gestellt, in denen Sie sich für eines von mehreren Waschmitteln entscheiden sollen.

Betrachten Sie bitte die folgende Tabelle und machen Sie sich mit Ihrem Aufbau vertraut. Sie finden darin drei Waschmittel mit den bereits bekannten Eigenschaften. Die Waschmittel wurden von einer unabhängigen Warentestung geprüft und die einzelnen Eigenschaften der Waschmittel bewertet. Den entsprechenden Bewertungsschlüssel finden Sie unter der Tabelle.

Eigenschaft {{ powder.name }}
{{ testCase }} {{ ($parent.$index == 0) ? (powder.testCaseRatings[$parent.$index] | number:2) : powder.testCaseRatings[$parent.$index] }}

Bewertungsschlüssel: ++ = sehr gut, + = gut, o = befriedigend, - = ausreichend, -- = mangelhaft


Bei Ihren kommenden Entscheidungen ist es wichtig, dass Sie diese anhand einer bestimmten Entscheidungsstrategie treffen. Im Folgenden wird Ihnen Ihre Entscheidungsstrategie erst allgemein und dann konkret beschrieben. Bevor es dann richtig los geht, können Sie sich mit der Anwendung dieser Strategie in zwei beispielhaften Entscheidungssituationen vertraut machen.
