Da die Ergebnisse des Fragebogens teilweise für eine zweite Untersuchung genutzt werden, für die einige Personen einen weiteren Test absolvieren, erstellen Sie bitte Ihren persönlichen Code.

Über den Code ergibt sich die Möglichkeit, die Ergebnisse beider Tests der gleichen Person zuzuordnen und für alle Teilnehmer die Anonymität zu wahren. Der persönliche Code besteht aus einer Kombination von Buchstaben und Zahlen, die außer Ihnen niemandem bekannt ist, der jedoch bei der zweiten Untersuchung wieder herleitbar ist.

Er setzt sich wie folgt zusammen:

  • Die ersten beiden Buchstaben des Vornamens Ihrer Mutter.
  • Die ersten beiden Buchstaben Ihres Geburtsortes.
  • Die ersten beiden Zahlen des Geburtstages Ihres Vaters.
  • Die ersten beiden Zahlen Ihres Geburtstages.

Beispiel:
Ihre Mutter heißt Helga, Sie sind in Frankfurt geboren, Ihr Vater ist am 16.8.1950 geboren, Sie sind am am 02.03.1980 geboren.

Ihr persönlicher Code lautet somit HEFR1602.
