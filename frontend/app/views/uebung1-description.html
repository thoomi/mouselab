Übungslauf 1

Auf der folgenden Seite werden Ihnen verschiedene Waschmittel gezeigt. Die Waschmittel unterscheiden sich hinsichtlich ihres Preises aber auch bezüglich anderer Eigenschaften. Sobald Sie sich für eines der Waschmittel entschieden haben, betätigen Sie einfach den „Button“ der sich unter diesem Waschmittel befindet. Oben rechts werden Sie sehen, wie lange Sie für Ihre Entscheidung benötigen. Denken Sie daran, sich so schnell aber auch so gut wie möglich für ein Waschmittel zu entscheiden.
